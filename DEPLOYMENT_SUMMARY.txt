================================================================================
QUICK START (30 MINUTES TO PRODUCTION)
================================================================================

1. Set up Google Cloud project
   gcloud init
   gcloud services enable run.googleapis.com cloudsql.googleapis.com

2. Create PostgreSQL database
   gcloud sql instances create voice-relay-db --tier db-f1-micro

3. Configure GitHub OAuth
   Visit: https://github.com/settings/developers
   Create new OAuth App
   Copy Client ID and Secret

4. Deploy backend
   cd backend
   gcloud builds submit --tag gcr.io/PROJECT_ID/voice-relay-backend
   gcloud run deploy voice-relay-backend

5. Test API
   curl https://YOUR_CLOUD_RUN_URL/health

6. Update mobile app
   Change API endpoint to your Cloud Run URL
   Build APK/IPA
   Submit to app stores

See DEPLOY_QUICK_START.md for detailed steps.
